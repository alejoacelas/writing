# Claude Code: how to fix a flaky Chrome connection

*Source: https://wow.pjh.is/journal/claude-code-fix-flaky-chrome-connection*  ·  *Published: 2026-04-03*

---

If the Claude-in-Chrome MCP connection is flaky, just re-install it:

```
claude extension install

```

**Update 2026-06-02:** This command no longer does anything.[1](#user-content-fn-1) Were I having trouble now, I'd try `/mcp`, select find Claude in Chrome, then "Reconnect". But I've not seen the flakiness issue recently, so I can't confirm that this helps.
