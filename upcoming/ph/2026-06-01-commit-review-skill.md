# Skill: turn a messy repo into clean commits

*Source: https://wow.pjh.is/journal/commit-review-skill*  ·  *Published: 2026-06-01*

---

Messy repo? Try my [/commit-review](https://github.com/HartreeWorks/skill--commit-review) skill:

> Analyse all uncommitted changes in the current repository, group them into logical changesets, and interactively commit each group after user approval.

Often, running many agents on `main` is faster than giving each its own worktree.
