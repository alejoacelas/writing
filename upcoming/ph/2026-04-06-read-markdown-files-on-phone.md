# How to read markdown files on your phone

*Source: https://wow.pjh.is/journal/read-markdown-files-on-phone*  ·  *Published: 2026-04-06*

---

When working with agents on my phone (whether via OpenClaw, Codex or Claude Code) I often end up with markdown files on my desktop computer that I want to read from my phone.

So I made [mdview](https://github.com/HartreeWorks/mdview), a utility that generates a temporary link to view those files, rendered in HTML, in my mobile browser.

Requires Tailscale.
