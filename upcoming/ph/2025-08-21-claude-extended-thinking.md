# Claude: enable extended thinking

*Source: https://wow.pjh.is/journal/claude-extended-thinking*  ·  *Published: 2025-08-21*

---

**By default, you should use Claude 4.1 Opus in "extended thinking" mode.**

More reasoning = better responses.

If you're using Opus, your question is probably important enough to warrant the extra wait (usually 10-60 seconds).

If you're a power user, you'll develop a sense of when it is _not_ worth enabling this, and instead just using Sonnet. Meantime, err on the side of having it enabled.

To enable it, click the chat settings button (second from the left):

Some examples where Sonnet is a better choice than Opus with extended thinking:

1. What does ngmi mean?
2. How to cancel an easyJet flight?
3. In Google Meet, can you make it default to always recording the call?

For questions (2) and (3), you should actually use ChatGPT: the web search is far better than Claude's.
