# Create Todoist tasks with voice input

*Source: https://wow.pjh.is/journal/todoist-create-tasks-voice*  ·  *Published: 2025-08-06*

---

Todoist can now [create tasks with voice input](https://www.todoist.com/help/articles/dictate-to-add-tasks-with-ramble-P1Raq7vVF).

On desktop, the shortcut is `Shift + Q`.
