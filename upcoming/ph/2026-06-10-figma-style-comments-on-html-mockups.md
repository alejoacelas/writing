# How to write Figma-style comments on HTML mockups

*Source: https://wow.pjh.is/journal/figma-style-comments-on-html-mockups*  ·  *Published: 2026-06-10*

---

When an agent makes a UI mockup or a HTML-formatted plan, I want to leave comments _à la_ Figma (click on element and type), then have the agent read those comments and do another round of work.

[Lavish](https://github.com/kunchenguid/lavish-axi) enables this.
