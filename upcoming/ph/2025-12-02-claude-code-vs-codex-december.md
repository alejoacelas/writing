# Coding in December: Claude Code for everything

*Source: https://wow.pjh.is/journal/claude-code-vs-codex-december*  ·  *Published: 2025-12-02*

---

With Opus 4.5, Claude Code is now clearly better than OpenAI Codex and Gemini CLI for nearly all coding tasks. It is smarter, faster and better at self-correction. It is also happy to run autonomously for longer—until the job is done.

Combine that with the better UI and additional features ([skills](https://wow.pjh.is/journal/claude-front-end-design-skill), [hooks](https://code.claude.com/docs/en/hooks-guide) and [all the rest](https://code.claude.com/docs/en/sub-agents)), and it's a no-brainer at the moment.[1](#user-content-fn-1)

For me, it's a step change. [Dan Shipper agrees](https://www.youtube.com/watch?v=F6O7r9LqhC0).

I've still not made time to try [Droid](https://factory.ai/product/ide).
