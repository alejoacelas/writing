# Skill: securely install MCP server

*Source: https://wow.pjh.is/journal/claude-skill-secure-mcp-install*  ·  *Published: 2025-12-28*

---

Many services have not yet released official MCP servers. There are lots of solid unofficial servers out there, but it'd be a very bad idea to use your [Google Workspace](https://wow.pjh.is/journal/claude-code-google-workspace-mcp) or [Slack](https://wow.pjh.is/journal/claude-skill-slack) credentials with code you don't trust _extremely_ highly.

I've made a [Claude Skill](https://github.com/HartreeWorks/claude-skill--secure-mcp-install) which helps you install MCP servers securely. Core functions:

1. Security audit before installing.
2. Disable automatic updates.

This setup, plus general heuristics, plus my solid understanding of IT security issues, means that the unofficial [Google Workspace MCP](https://wow.pjh.is/journal/claude-code-google-workspace-mcp) passes the cost:benefit for me.
