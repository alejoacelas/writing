# Coding in October: Claude Code for frontend, Codex for everything else

*Source: https://wow.pjh.is/journal/claude-codex-vs-codex*  ·  *Published: 2025-10-14*

---

Claude Code with Sonnet 4.5 is fast. Codex with GPT-5 high is slower, but smarter.

For now, I'm using Claude Code for UI components, and Codex for everything else (backend, refactoring, bugfixes, etc).

Twitter mostly agrees with me on this. In fact, there's unusually high consensus at the moment.

So, I think this is the correct approach—this month. [1](#user-content-fn-1) [2](#user-content-fn-2)
