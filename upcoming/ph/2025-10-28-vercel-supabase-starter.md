# Starter template for building a web app with user authentication, database and a decent UI

*Source: https://wow.pjh.is/journal/vercel-supabase-starter*  ·  *Published: 2025-10-28*

---

Want to build a web app? The [Vercel Supabase Starter](https://vercel.com/templates/next.js/supabase) is excellent.

Pair it with [AI SDK](https://ai-sdk.dev/) and a component library like [Shadcn](https://ui.shadcn.com/) or [DaisyUI](https://daisyui.com/).

Together, these are a great foundation for vibe-coding web apps with Codex or Claude.

You'll need some coding experience—or time learn with an LLM. If you want to skip that, try [Replit](https://replit.com/).
