# Claude can now search and send Slack messages

*Source: https://wow.pjh.is/journal/claude-slack-connector*  ·  *Published: 2026-02-25*

---

Claude's [Slack connector](https://claude.com/connectors/slack) can now read, search and send Slack messages.

To enable it: go to [claude.ai/settings/connectors](https://claude.ai/settings/connectors).

This is separate from the [Claude Slack app](https://claude.com/claude-for-slack), which adds a Claude bot you can @mention.
