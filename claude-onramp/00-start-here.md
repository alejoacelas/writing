# Start here

This guide gives Claude your files, your apps, and — by the end — your whole
computer, so you can hand off work instead of doing it. Not asking a chatbot a
question and copying the answer out: handing over a task and getting back
finished work.

That's worth doing because it changes which tasks you take on at all. The ones
you skip today because they're too technical, too fiddly, too slow, or too vague
to start — those become things you delegate in a sentence. This guide is about
getting to that point.

## The two tools

**Claude Cowork** works out of the box. Install one app, connect your accounts,
start delegating. No terminal, nothing to configure. It's the right place to
begin.

**Claude Code** does more — it can run any tool your computer can, reach the
whole internet, and work on its own for an hour at a stretch — in exchange for
about thirty minutes of setup. It's where you'll end up for anything serious.

## Which to read

**Start with Cowork** (Part 1). It teaches the four ideas Claude Code takes for
granted — working in a folder, choosing a model, granting permissions, and
managing context — while you install nothing but one app. Then Part 2 graduates
you to Claude Code.

**Already fluent** with agents in a folder, models, permissions, and context
windows? Skip straight to [Part 2: Claude Code](02-claude-code.md).

## One choice before you start

**Which account.** Use your organization's **Claude Team account** if you have
one; a personal **Max** plan works just as well. Both include everything in this
guide. → [Team vs. personal, and which tier →](decisions/account.md)
